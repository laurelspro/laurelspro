## What we build

Laurels is a verified professional profile platform for **product managers, UX designers, data analysts, DevOps engineers, solutions architects** — people who work in tech but don't need a developer portfolio.

We give them something better than a resume:

- **Verified identity** — name confirmed against payment card. No fake accounts.
- **Live credentials** — GitHub stats, projects, and tools that update in real time.
- **Embeddable widget** — drop your profile into any website with one line of code.
- **Video intro** — record up to 3 minutes directly in the browser.
- **Real analytics** — see who viewed your profile, where they came from, and whether they reached out.
